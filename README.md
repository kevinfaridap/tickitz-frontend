## About This Project

Tickitz is an online ticket ordering service application that informs users to see movies that are now showing and upcoming movies. This application helps users to orders ticket quickly, effectively and easily. This application was built by ReactJs, NodeJs, and ExpressJs.



### Built With

* [ReactJs](https://reactjs.org/)
* [Bootstrap V.4](https://getbootstrap.com/)
* **JWT** for Authentication
* **Nodemailer** for Mailer



<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them

* [nodejs](https://nodejs.org/en/download/)


### Installation

1. Clone the repo
```sh
git clone https://github.com/kevinfaridap/week5-frontend-beginner.git
```
2. Install NPM packages
```sh
npm install
```
3. Add .env file at root folder project, and add following
```sh
REACT_APP_API=[Backend API]
REACT_APP_SOCKET=[Backend API]
```
